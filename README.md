# ts-minimal-kit-cli

CLI installer for TypeScript minimal kits:
[ts-minimal-kit](https://github.com/k1de/ts-minimal-kit)
and
[ts-minimal-kit-web](https://github.com/k1de/ts-minimal-kit-web)

## Install

```bash
npm install -g ts-minimal-kit-cli
```

## Usage

### Minimal TypeScript kit

```bash
ts-minimal-kit my-project
```

Strict typing, clean setup, ready to code.

### Minimal TypeScript web kit

```bash
ts-minimal-kit-web my-project
```

Zero dependencies, strict typing, full-stack ready.

## What happens

Both commands will:

-   Clone the template
-   Initialize new repository
-   Install dependencies
-   Update TypeScript to latest
-   Open in VS Code (if available)

## Requirements

-   Node.js >= 18.0.0
-   Git
-   npm
